About Recipes Diary🍲

This is a recipe management web application built using Angular and Firebase.

  • 🔐 User Authentication using Firebase
  • 📖 Add, Edit, Delete Recipes
  • 🔍 Search Recipes
  • ❤️ Favorite Recipes (User-specific)
  • ☁️ Data stored in Firestore

This project demonstrates full-stack development with real-time database integration.
