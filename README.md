# 🍲 Recipes Diary

A modern recipe management web app built using Angular and Firebase.

---

## 🚀 Features

- 🔐 Firebase Authentication (Login / Logout)
- 📖 Add, Edit, Delete Recipes
- 🔍 Real-time Search
- ❤️ Favorite Recipes (User-specific)
- ☁️ Firestore Database Integration
- 🎨 Clean & Modern UI

---

## 🛠 Tech Stack

- Angular
- Firebase (Auth + Firestore)
- TypeScript
- SCSS

---

## ⚙️ Run Locally

```bash
npm install
ng serve
```
